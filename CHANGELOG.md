## [2.0.1] - 2026-03-24

### 改进
- EDC 站点添加工具全面优化：处理逻辑移至后台线程，UI 不再阻塞。
- 新增「一键录制」坐标模式：屏幕顶部浮动引导条，连续点击 7 个按钮即可完成坐标配置；支持双击表格行单步重录。
- 新增「回放测试」功能：依次在各坐标位置闪烁标记，便于确认位置是否正确。
- 新增进度条，实时显示处理行数与百分比。
- 新增重试机制，点击操作失败后可自动重试（可配置重试次数与步骤间隔）。
- 新增日志「清空」与「导出」按钮。
- ESC 监听从轮询改为事件驱动，降低 CPU 占用。

## [2.0.0] - 2026-03-04

### 新增
- 新增 `仕样书工作流工具` 页面（`spec_workflow_page.py`），在单一界面内切换三种流程：生成 Data Set / 数据清洗 / Codelist 处理。

### 改进
- 文件格式转换工具扩展为四模式统一入口：CSV 转 XLSX、XLSX 转 CSV（UTF-8 BOM）、CSV 转 UTF-8（BOM）、CSV 引号去除。
- 主界面工具入口精简为 9 个，移除重复入口并统一命名与描述文案。
- 清理已废弃页面代码：`data_cleaner_page.py`、`codelist_processor_page.py`、`xlsx_restructure_page.py`、`csv_quote_remover_page.py`。
- 打包配置（`tools_box.spec`、`packaging/pyinstaller/tools_box_onedir.spec`）同步隐藏导入，移除已删除页面引用。
- 全量文档统一升级到 `v2.0.0`：`README.md`、`CHANGELOG.md`、`docs/*.md`、`packaging/README.md`、`AGENTS.md`。

## [1.8.2] - 2026-03-03

### 新增
- `DataMaskingService` 新增替换开关回归测试：当关闭人名/施设替换时，DM 对应字段保持原值输出。

### 改进
- 统一外部配置文件策略：`EDC 站点添加` 与 `数据模糊化 Pattern1` 合并为单一配置文件 `tools_box_config.json`，固定保存到系统用户配置目录，并支持从旧独立配置文件自动迁移。
- 数据模糊化 DM 专属设置界面重构：人名/施设字段与替换值改为同行展示，提升配置可读性。
- DM 专属设置新增两组独立启用开关（默认勾选）：取消勾选后即使字段非空也不执行替换，直接输出原值。
- DM 专属设置布局对齐优化：首个输入框与标准表单列对齐、替换值标签右移、尾部 `启用替换` 右对齐。
- 步骤 3 内普通说明文本统一为蓝色（`dmFieldHint`），与红色警示（`dmAlertNote`）视觉分层更清晰。
- 文档同步更新到 v1.8.2：`README.md`、`docs/用户手册.md`、`docs/API文档.md`、`docs/开发文档.md`、`docs/XLSX文件制作规范.md`、`docs/UI界面规范.md`。

## [1.8.1] - 2026-02-10

### 改进
- 数据模糊化页面与服务层完成一轮代码精简：删除冗余分支，统一 USUBJID 预览与执行阶段的转换逻辑，降低规则漂移风险。
- `DataMaskingService` 移除未被调用的 legacy 兼容 API（`set_baseline_from_dm`、`process_file` 及其相关方法），对外保留 `scan_pattern1` / `run_pattern1` 作为主流程接口。
- 步骤一文件导入路径统一：按钮选择与拖拽导入共用同一入列逻辑，减少重复失效处理代码。
- 数据模糊化相关文档（`README.md`、`docs/API文档.md`、`docs/开发文档.md`、`docs/用户手册.md`、`docs/XLSX文件制作规范.md`）同步更新到 v1.8.1。

## [1.8.0] - 2026-02-09

### 新增
- 抽离 EDC 站点添加服务层：新增 `EdcSiteAdderService`，页面逻辑与自动化执行逻辑解耦。

### 改进
- 统一命名规范：GUI 页面文件统一为 `*_page.py`，业务实现统一为 `*_service.py`，提升模块可读性与可维护性。
- 清理历史兼容别名与未使用导入，减少冗余代码和维护成本。
- 主界面卡片布局优化为固定卡片宽度 + 自适应列数，窗口拉伸时不再仅放大卡片间距。
- 全量同步文档（`README.md`、`docs/API文档.md`、`docs/开发文档.md`）到最新目录结构和接口命名。

## [1.7.0] - 2026-01-27

### 新增
- 死链检测工具：支持检测单个 HTML 文件或文件夹，生成报告，支持基础 URL 与超时设置。
