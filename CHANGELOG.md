# 更新日志

所有对项目的显著更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [Unreleased]

### 新增
- 死链检测工具：支持检测单个 HTML 文件或文件夹，生成报告，支持基础 URL 与超时设置。

### 改进
- 主窗口重构为分类卡片布局，支持滚动浏览与更清晰的工具分组。

## [1.6.0] - 2025-12-09

### 新增
- 文件格式转换工具：单一窗口支持 CSV 转 XLSX、XLSX 转 CSV(UTF-8 BOM) 与 CSV 转 UTF-8(BOM) 三种模式。
- 获取文件字段工具：批量读取 CSV/XLSX 字段并输出汇总 CSV。

### 改进
- 文件格式转换模式切换使用分段按钮样式，并自动过滤不匹配文件。
- 更新版本信息与界面文案。

## [1.5.0] - 2025-07-23

### 新增
- 数据模糊化工具：支持 SDTM 数据集脱敏与基准 USUBJID 过滤。
- CSV 引号去除工具：智能清理多层引号并批量处理。
- 全角转半角转换工具：支持 Excel 文件中的字符转换。
- 生成 Data Set 工具增强：支持 STUDYID 选择与仕样书过滤。

### 改进
- 优化拖拽文件处理与平台目录获取逻辑。
- 数据模糊化输出编码统一为 UTF-8-SIG。

## [1.4.0] - 2025-05-12

### 新增
- Codelist 处理工具：编码映射与多语言值转换。

### 改进
- 优化日期列判断逻辑与处理性能。

## [1.3.0] - 2025-04-23

### 新增
- 数据清洗工具：基于仕样书规则进行患者过滤、列裁剪与行过滤。

## [1.0.0] - 2025-04-08

### 新增
- 基础工具箱框架。
- 日期转换工具。
- CSV 转 XLSX 工具。
- XLSX 转 CSV 工具。
- EDC 站点添加工具。
- 生成 Data Set 工具。
