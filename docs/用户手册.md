## 日期转换工具

### 功能说明
支持常见日期格式自动识别并转换为 ISO 8601（`YYYY-MM-DD`）。

### 使用方法
1. 点击主界面的“日期转换”按钮。
2. 在左侧输入日期文本。
3. 在右侧查看转换结果。

### 注意事项
- 建议输入完整日期。
- 无法识别时会显示提示信息。

## 文件格式转换工具

### 功能说明
单一页面切换 4 种模式：
- CSV 转 XLSX
- XLSX 转 CSV（UTF-8 BOM）
- CSV 转 UTF-8（BOM）
- CSV 引号去除

### 使用方法
1. 点击“文件格式转换”。
2. 在“转换模式”中选择需要的模式。
3. 选择文件或文件夹（支持拖拽，列表会按模式自动过滤扩展名）。
4. 可选设置输出目录。
5. 点击“开始转换”。

### 说明
- `CSV 转 UTF-8(BOM)` 与 `CSV 引号去除` 默认覆盖原文件；设置输出目录后会另存。
- 模式切换时，不符合后缀的文件会自动从列表移除。

## 工作表拆分工具

### 功能说明
将一个 Excel 文件按工作表拆分成多个 CSV（UTF-8 BOM）。

### 使用方法
1. 点击“工作表拆分”。
2. 选择一个 `.xlsx` 文件。
3. 可选设置输出路径。
4. 点击“开始拆分”。

### 注意事项
- 工作表名会用于输出文件名，非法字符会替换为 `_`。
- 同名冲突会自动追加序号。

## 获取文件字段工具

### 功能说明
批量提取 CSV / XLSX 文件表头字段并输出汇总报表。

### 使用方法
1. 点击“获取文件字段”。
2. 选择目标文件夹（支持拖拽）。
3. 按需勾选“包含子文件夹”，设置“列名行”。
4. 点击“开始提取字段”。

### 输出说明
- 默认输出到所选目录，文件名为 `file_fields_summary.csv`（若重名自动加序号）。

## 死链检测工具

### 功能说明
检测 HTML 文件中的链接状态，支持单文件或文件夹批量扫描。

### 使用方法
1. 点击“死链检测”。
2. 选择 HTML 文件或文件夹。
3. 可选设置基础 URL、超时、是否递归子目录。
4. 点击“开始检测”。

### 输出说明
- 生成 `dead_link_report_*.txt` 报告，包含正常/失败/超时/跳过统计。

## 仕样书工作流工具

### 功能说明
统一入口切换三种依赖仕样书的流程：
- 生成 Data Set
- 数据清洗
- Codelist 处理

### 使用方法
1. 点击“仕样书工作流”。
2. 在顶部选择处理模式。
3. 上传对应仕样书。
4. 选择输入文件或文件夹（支持拖拽，按模式限制后缀）。
5. 可选设置输出路径。
6. 点击“开始处理”。

### 模式说明
- 生成 Data Set：输入 `.xlsx`，可配置 `STUDYID`。
- 数据清洗：输入 `.csv`，按 Patients/Process/Files 规则筛选。
- Codelist 处理：输入 `.csv`，按 Process/CodeList/Files 做映射转换。

### 注意事项
- 三种模式会分别记住自己的文件列表、仕样书和输出目录。
- 必须先加载仕样书，才能执行处理。

## 数据模糊化工具

### 功能说明
提供 SDTM Pattern1 脱敏处理流程（扫描 + 执行）。

### 使用方法
1. 点击“数据模糊化”。
2. 在 Pattern1 中导入 CSV 文件集（必须包含且仅包含一个 `DM.csv`）。
3. 先执行扫描，确认字段分布与日期识别结果。
4. 配置 USUBJID、STUDYID、DM 专属替换规则等参数。
5. 执行处理并查看结果。

### 注意事项
- 处理前必须扫描。
- 若 DM 不唯一或缺少关键字段，流程会中止并提示。

## EDC站点添加工具

### 功能说明
通过自动化操作辅助执行 EDC 站点批量添加。

### 使用方法
1. 准备站点信息 Excel。
2. 打开并登录目标 EDC 页面。
3. 进入“EDC站点添加”页面，确认配置参数。
4. 点击“开始处理”。
5. 可按 `ESC` 终止。

### 注意事项
- 执行时避免手动干预鼠标键盘。
- 坐标参数需与当前分辨率和页面布局匹配。

## 全角转半角工具

### 功能说明
将 Excel 文本中的全角字符统一转换为半角字符。

### 使用方法
1. 点击“全角转半角”。
2. 选择 Excel 文件。
3. 可选设置输出路径。
4. 点击“开始转换”。

### 转换范围
- 全角数字、英文字母、符号和全角空格。

## 常见问题

### Q: 拖拽文件没有反应？
A: 请确认文件后缀符合当前模式；不符合的文件会被过滤。

### Q: 输出文件在哪？
A: 未设置输出目录时默认输出到输入文件所在目录。

### Q: 为什么提示必须先上传仕样书？
A: 仕样书工作流三种模式都依赖仕样书规则，未加载时不可执行。

### Q: 处理失败怎么排查？
A: 先看页面状态提示和弹窗错误信息，再检查输入文件格式和关键字段是否齐全。
