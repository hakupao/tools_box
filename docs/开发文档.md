### 设计模式

#### 分层架构
- **Service (utils/)**: 业务逻辑和数据处理
- **View/Page (gui/widgets/)**: 用户界面展示与交互
- **Main Window (gui/main_window.py)**: 页面注册与导航调度

#### 组件化设计
每个工具都采用独立的组件设计：
- 页面类（`*_page.py`）：负责 UI 展示和用户交互
- 服务类（`*_service.py`）：负责具体业务逻辑
- 工具函数：提供通用的辅助功能

## 核心技术

### GUI 框架

#### PySide6 + QFluentWidgets
- **优势**: 现代化桌面 UI、组件丰富、体验一致
- **用法**: 页面继承 `QWidget`，主窗口继承 `FluentWindow`
- **样式**: 统一 Win11 Fluent 风格
- **拖拽**: 使用 Qt 原生拖拽事件（`dragEnterEvent`/`dropEvent`）

### 数据处理

#### Pandas
- **用途**: CSV/Excel文件读写和数据处理
- **配置**: 统一使用 `dtype=str` 保持数据格式
- **编码**: 使用 `utf-8-sig` 处理中文BOM

#### OpenPyXL
- **用途**: Excel文件的深度操作
- **特性**: 支持格式保持和样式处理

### 网络与解析

#### Requests + BeautifulSoup
- **用途**: HTML 链接检测与解析
- **配置**: 使用 Session + HEAD/GET 双策略
- **输出**: 生成文本报告并分类统计

### 自动化控制

#### PyAutoGUI
- **用途**: 鼠标键盘自动化控制
- **安全**: 设置 `FAILSAFE = True`
- **配置**: 可调节的点击坐标和延时

## 添加新工具指南

### 1. 创建服务类

在 `src/utils/` 中创建服务类：

```python
# src/utils/new_tool_service.py
class NewToolService:
    """新工具服务"""

    def process_file(self, input_file: str, output_path: str | None = None) -> tuple[bool, str]:
        try:
            # 实现具体处理逻辑
            return True, ""
        except Exception as exc:
            return False, str(exc)
```

### 2. 创建页面组件

在 `src/gui/widgets/` 中创建页面：

```python
# src/gui/widgets/new_tool_page.py
from PySide6.QtWidgets import QWidget
from ...utils.new_tool_service import NewToolService

class NewToolPage(QWidget):
    def __init__(self, main_window) -> None:
        super().__init__()
        self.main_window = main_window
        self.service = NewToolService()
```

### 3. 注册到主窗口

修改 `src/gui/main_window.py`：

```python
from .widgets.new_tool_page import NewToolPage

TOOL_PAGE_REGISTRY = [
    # ... existing pages
    ("new_tool", NewToolPage, SOME_ICON, "新工具"),
]
```

### 4. 更新版本与文档

更新 `src/version.py`、`CHANGELOG.md` 以及相关文档说明。

## 常见开发问题

### 1. 中文编码问题
**解决方案**: 统一使用 `utf-8-sig` 编码

### 2. 文件路径问题
**解决方案**: 使用 `pathlib` 处理路径，避免硬编码

### 3. 内存占用问题
**解决方案**:
- 分批处理大文件
- 及时释放不用的对象
- 使用生成器处理大数据集

### 4. GUI 响应问题
**解决方案**:
- 使用线程处理耗时操作
- 及时更新进度显示
- 提供取消操作功能

## 代码规范

### Python 代码风格

遵循 PEP8 规范：
- 使用 4 个空格缩进
- 行长度限制为 100 字符
- 类名使用 PascalCase
- 函数名使用 snake_case
- 常量使用 UPPER_CASE

### 文档字符串

使用中文文档字符串：
```python
def process_file(self, input_file: str, output_path: str = None) -> bool:
    """
    处理单个文件
    
    Args:
        input_file: 输入文件路径
        output_path: 输出路径，可选
        
    Returns:
        bool: 处理是否成功
    """
```

### 错误处理

统一的错误处理模式：
```python
try:
    # 主要逻辑
    result = process_data()
    return True, result
except FileNotFoundError as e:
    return False, f"文件未找到: {str(e)}"
except Exception as e:
    return False, f"处理失败: {str(e)}"
```
